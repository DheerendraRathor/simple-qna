# Simple QnA

# To install it
- Install python 2.7
- Run `pip install -r requirements.txt`
- Run `pip install -r requirements_sec.txt`

# To run it
- `python manage.py runserver`
- Open http://localhost:8000

- root:root12345 for admin panel
